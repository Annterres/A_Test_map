# A_Test_map Triggers

## Structure
- `skills/` - Custom hero abilities
- `system/` - Core systems (hero level init, etc.)
- `shops/` - Shop mechanics

## Global Variables (Ctrl+B in YDWE)
- `BM_Hash` (哈希表 / hashtable) - Skill data storage
- `BM_Bolt` (闪电 / lightning) - Lightning bolt visual

## Custom Object IDs
- `A000`: 雷霆之怒 (parent: AHfs Fan of Knives, hotkey: W)
- `A001`: 八面龙卷风 (parent: AEfk Breath of Fire, hotkey: Q)
- `I000`: 免费的恢复卷轴 (parent: sreg)
- `I001`: 免费的魔法卷轴 (parent: sman)
- `h000`: Custom shop (parent: hvlt)
